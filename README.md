# PHPブログシステム

## 概要

Markdown記法を用いた.mdファイルを作成するだけでブログ記事として投稿が完了するお手軽なシステム。サイト検索やタグ、シェアなど基本機能は網羅

## フォルダ構成

---

phpBlogSystem

┣━ css

┃    ┗━━ <u>style.css</u>



┣━images

┃    ┗━━ background.jpg (SNSシェア時のデフォルト背景画像)

┣━text (ブログ記事フォルダ。.mdを作ると1ファイル1記事になる)

┣━article.php(記事表示メイン)

┣━functions.php(ブログの根幹システム)

┣━index.php(ブログのホームページ)

┣━list.php(記事一覧表示)

┣━sidebar.php(サイドバー表示)

┗━ readme.md(このファイル)

---

## 使用前設定

1. /functions.phpをテキストエディタで開き、743行目定数・設定を変更
   最低限いじるのはブログタイトルとブログ説明のみ。必要に応じて他も変更可

2. functions.phpを開き、526行目のデフォルト画像のパスを絶対パスで指定
   ここで指定した画像がSNSシェアに使われます(記事内に動画や画像が挿入されている場合はそちらを優先)

## ブログへのアクセスと更新

ファイルを全てサーバーにアップロードし、index.phpへ

記事の更新はローカルで行います。/text/フォルダ内の.mdファイルを修正または更新でOK

## 記事の仕様

1. 記事内の最初の見出し1「#」がそのまま記事名になります

2. カテゴリーを設定するには記事内に「Tags:カテゴリー名」と記載する
